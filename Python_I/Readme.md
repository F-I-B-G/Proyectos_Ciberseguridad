# Proyectos Básicos de Ciberseguridad con Python

Este repositorio reúne **proyectos introductorios** que utilizan **Python como lenguaje core**, con el objetivo de:

- **Asentar conocimientos de programación en Python** desde lo más básico hasta estructuras más avanzadas.  
- **Aplicar esos conocimientos en el ámbito de la ciberseguridad**, construyendo scripts prácticos y reutilizables.  
- **Combinar teoría y práctica** para fortalecer tanto las bases del lenguaje como los fundamentos de seguridad informática.  

## Contenido

- **Scripts básicos** → primeros pasos en Python aplicados a seguridad.  
- **Redes y protocolos** → parsing de tráfico, validación de respuestas y simulaciones controladas.  
- **Esteganografía y ocultamiento de datos** → ejemplos simples de codificación y decodificación.  
- **Cifrado y hashing** → prácticas introductorias de protección de información.  
- **Automatización** → pequeños scripts para escaneo y generación de reportes.  

## Objetivo

Construir una base sólida de proyectos que permitan **aprender Python desde la perspectiva de la ciberseguridad**, paso a paso, con ejemplos claros y aplicables.
